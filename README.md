# LoomBrain Plugins

Official Claude Code plugins for [LoomBrain](https://loombrain.com) — a personal knowledge graph SaaS.

## Installation

```bash
/plugins marketplace add amentistudio/loombrain-plugins
```

## Available Plugins

### loombrain-sessions

Auto-capture Claude Code sessions to LoomBrain episodic memory. Every meaningful coding session is automatically saved when you exit or `/clear`, building a searchable knowledge base of your development work.

```bash
/plugin install loombrain-sessions
```

See [plugin README](plugins/loombrain-sessions/README.md) for setup and configuration.
